# HeatSense 🌡️ — Urban Heat Island Intelligence Platform

AI-powered real-time urban heat monitoring with Claude-driven insights.

## Features
- Live weather data for any city via OpenWeatherMap API
- Claude AI heat risk analysis with actionable recommendations  
- 5-day heat forecast with visual trend bars
- City vs City heat comparison with winner detection
- Dynamic heat advisory panel (CRITICAL/HIGH/MODERATE/LOW)
- Real-time thermal heatmap visualization
- Wind speed, humidity, heat index, risk level metrics

## Tech Stack
Next.js 15 · TypeScript · Tailwind CSS · Claude API · OpenWeatherMap API · Vercel

## Impact
Addresses SDG 11 (Sustainable Cities) and SDG 13 (Climate Action) by giving citizens 
and city planners real-time heat intelligence to reduce heat-related illness and death.
